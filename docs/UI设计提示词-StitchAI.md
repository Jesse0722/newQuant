# 量化交易系统 - UI 线框图

> 左侧固定侧边栏（4 块导航）+ 右侧主内容区。风格：专业量化看板，深色/浅色主题可切换。
>
> 基于 `01-产品设计-定稿.md`（2026-03 版），共 6 个页面。

---

```

```
