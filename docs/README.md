# 量化交易系统 - 文档目录

本目录存放产品设计与实施计划，纳入版本管理。

| 文件 | 说明 |
|------|------|
| `01-产品设计-定稿.md` | 产品设计定稿版（定位、模块、技术架构、API、数据模型） |
| `02-脑暴续-待定稿.md` | 待进一步脑暴与拍板的细化项（页面结构、因子模板、UI 交互、回测引擎等） |
| `03-实施计划-Phase1.md` | Phase 1 具体实施步骤 |
| `PLANNING_WITH_SUPERPOWERS.md` | 使用 Superpowers 进行问答式规划的说明 |
| `plans/` | 设计文档输出目录（Superpowers brainstorming 保存路径） |

**使用建议**：
1. 安装 Superpowers：在 Cursor Agent 聊天中输入 `/plugin-add superpowers`
2. 用问答式规划完成 `02-脑暴续-待定稿.md` 中的拍板，将结论合并进 `01-产品设计-定稿.md`
3. 再按 `03-实施计划-Phase1.md` 推进开发
